# Deploy backend to Render — 5 min

The PWA already lives on Firebase Hosting. The Express server (`server.js`) handles payments, OTP, reports, and admin actions — it needs its own host. Render's free tier is fine to start.

## Steps

### 1. Push code to GitHub
```bash
cd /Users/moses/Developer/nodalali
git init && git add -A && git commit -m "initial"
gh repo create nodalali --public --source=. --push
```
(or use github.com/new manually if you don't have `gh`)

### 2. Sign up at Render
https://render.com → "Get Started" → use GitHub login.

### 3. New Blueprint
- Render dashboard → **New +** → **Blueprint**
- Connect the `nodalali` repo
- Render reads `render.yaml` automatically and creates the `nodalali-api` service

### 4. Fill in secret env vars
Render will ask for the values marked `sync: false` in `render.yaml`:

| Variable | Value |
|---|---|
| `SUPABASE_URL` | (from Supabase dashboard → Settings → API) |
| `SUPABASE_ANON_KEY` | (from Supabase dashboard → Settings → API → "anon public") |
| `SUPABASE_SERVICE_ROLE_KEY` | (from Supabase dashboard → Settings → API → "service_role" — KEEP SECRET) |
| `ZENOPAY_API_KEY` | (after you sign up at zenoapi.com) |
| `ZENOPAY_WEBHOOK_SECRET` | (optional) |
| `BEEM_API_KEY` / `BEEM_SECRET` | (after you sign up at beem.africa) |
| `BREVO_API_KEY` | (after you sign up at brevo.com) |

`ADMIN_TOKEN` and `JWT_SECRET` will be auto-generated by Render.

### 5. Deploy
Click **Apply** → Render builds and starts the server (~3 min). You'll get a URL like `https://nodalali-api.onrender.com`.

### 6. Point the PWA at the live API
Edit `script.js` — find any `fetch('/api/...)` calls and prepend the Render URL when running on Firebase Hosting:

```js
const API_BASE = location.hostname.includes('web.app')
  ? 'https://nodalali-api.onrender.com'
  : '';
fetch(API_BASE + '/api/listings/check', ...)
```

(I can do this step automatically once the Render URL is live — just tell me the URL.)

### 7. Re-deploy frontend
```bash
cp script.js public/ && firebase deploy --only hosting
```

## Free-tier caveat
Render's free web service **sleeps after 15 min idle** and takes ~30s to wake up on first request. For a marketplace this is acceptable early on. Upgrade to $7/mo Starter when you have paying customers.
